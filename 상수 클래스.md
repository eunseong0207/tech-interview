# Flutter 상수 클래스란?
앱 전체에서 반복적으로 사용되는 고정된 값(문자열, 색상, 크기,스타일 등)을 한곳에서 관리하기 위해 설계된 클래스

## 상수 클래스의 주요 특징
1. static 키워드를 사용하여 클래스 이름으로 접근
2. 클래스의 객체를 생설할 필요가 없고, 생성자를 private으로 선언하여 불필요한 메모리를 사용하지 않음
3. 앱의 어느곳에서든 클래스명.변수명 형태로 접근 가능

### 장점
1. 메모리 최적화
2. 가독성 향상
3. 메모리 최적화

### 사용 예시
```dart
import 'package:flutter/material.dart';

class AppConstants {
  // 1. 인스턴스화 방지 (생성자를 private으로 숨김)
  AppConstants._();

  // 2. 색상 상수
  static const Color primaryColor = Color(0xFF6200EE);
  
  // 3. API 엔드포인트 상수
  static const String baseUrl = 'https://api.myservice.com/v1';
  
  // 4. 레이아웃 치수 상수
  static const double defaultPadding = 16.0;
}

// 실제 사용 (위젯 내부)
class MyWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Container(
      // 하드코딩 없이 상수 클래스 참조
      padding: const EdgeInsets.all(AppConstants.defaultPadding),
      color: AppConstants.primaryColor,
    );
  }
}
```
