# Flutter 상수 클래스란?
앱 전체에서 반복적으로 사용되는 고정된 값(문자열, 색상, 크기,스타일 등)을 한곳에서 관리하기 위해 설계된 클래스
const 키워드를 사용하여 선언

### 상수클래스 장점
1. 유지보수 용이
- 상수 클래스 파일 하나만 열어서 고치면 앱 전체가 동시에 바뀜
2. 가독성 향상
- 클래스명.변수명 형태로 사용할 수 있음
3. 메모리 최적화
- 앱 시작시 메모리에 한번만 할당되기에 메모리 사용량이 줄어듬

### 상수클래스 단점
1. 동적인 대응 불가
- 앱 실행 중에 값이 변해야 하는 상황에서는 해당 방식을 쓸 수 없음
2. 관리소홀
- 모든 상수를 한 파일에 다 때려박으면 나중에 관리가 불가능한 파일이 될 수 있음

### 사용 예시
```dart
import 'package:flutter/material.dart';

class AppConstants {
  // 1. 인스턴스화 방지 (생성자를 private으로 숨김)
  AppConstants._();

  // 2. 색상 상수
  static const Color primaryColor = Color(0xFF6200EE);
  
  // 3. API 엔드포인트 상수
  static const String baseUrl = 'https://api.myservice.com/v1';
  
  // 4. 레이아웃 치수 상수
  static const double defaultPadding = 16.0;
}

// 실제 사용 (위젯 내부)
class MyWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Container(
      // 하드코딩 없이 상수 클래스 참조
      padding: const EdgeInsets.all(AppConstants.defaultPadding),
      color: AppConstants.primaryColor,
    );
  }
}
```
